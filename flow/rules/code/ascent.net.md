# Ascent.net

#### Ascent.net.isIp

Checks if a string is a valid IP address (IPV4 or IPV6).

<pre><code><strong>ascent.net.isIp(input: string): boolean;
</strong></code></pre>

| Parameter | Type   | Description                  |
| --------- | ------ | ---------------------------- |
| Input     | String | IP address value (IPV4/IPV6) |
