# Search

## Help

```bash
#./apicactl s -h
NAME:
   apicactl search - search sudo

USAGE:
   apicactl search command [command options] [search_term, relative time]

COMMANDS:
   next, n  query n
   help, h  Shows a list of commands or help for one command

OPTIONS:
   --st value      Relative start time. (default: "10h")
   --et value      Relative end time. (default: "10h")
   --filter value  --filter 'Hostname=127.0.0.1,10.231.253.255;Message=tito*'
   --help, -h      show help (default: false)
   --version, -v   print the version (default: false)

```
